# 8.渐变动画

##### 1.

```objc
    // 开始动画
    [UIView beginAnimations:nil context:nil];
    // 设置动画时间
    [UIView setAnimationDuration:1.0f];
    
    // 2.动画代码
    CGRect frame = self.animationView.frame;
    frame.origin.y -= 50;
    self.animationView.frame = frame;
    
    // 3.提交动画
    [UIView commitAnimations];
```
